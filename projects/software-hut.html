Projects - "Software Hut" (University Project)

System for organising GCSE Students' Work Experience Placements

In the Software Hut module at the University of Sheffield, we were tasked with producing software for a real client (a university department or a local charity, for example). We were assigned to build a system to handle applications for their work experience program

The system was a web app, built in Ruby on Rails, which allowed the client to create and modify information pages on the website in a CMS-like interface. It would also let them manage the applications process for the work experience program, with the ability to create and modify different versions of the application form (meaning questions could be changed without deleting applications) as well as viewing the results they would generate.
